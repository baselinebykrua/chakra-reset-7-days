# Chakra Reset 7 Days - Design Brainstorm

## Context
A 7-day chakra healing program website that guides users through daily meditation and exercises. The site needs to be calming, easy to navigate, and clearly organized by day. Users should feel welcomed into a healing journey.

---

## Design Approach: Wellness Minimalism with Spiritual Depth

**Design Movement:** Modern Wellness Design + Spiritual Minimalism
- Inspired by contemporary meditation apps (Calm, Headspace) but with deeper spiritual grounding
- Clean, breathing layouts that evoke tranquility
- Intentional use of space and color to guide emotional states

**Core Principles:**
1. **Breathing Space** - Generous whitespace and padding to create calm, non-cluttered experience
2. **Spiritual Authenticity** - Chakra colors and symbols used meaningfully, not decoratively
3. **Progressive Disclosure** - Information revealed gradually as user progresses through days
4. **Embodied Interaction** - Subtle animations that feel like energy flowing through the interface

**Color Philosophy:**
- **Background:** Soft cream/off-white (almost white but warm) - creates safe container
- **Chakra Accents:** Each day features its chakra color (Root: Red, Sacral: Orange, Solar Plexus: Yellow, Heart: Green, Throat: Blue, Third Eye: Indigo, Crown: Violet)
- **Text:** Deep charcoal for readability against light backgrounds
- **Accents:** Muted gold/bronze for highlights and CTAs - suggests spiritual value

**Layout Paradigm:**
- Vertical scroll-based journey (mirrors the chakra journey from root to crown)
- Each day gets its own "card" or "section" with breathing room
- Left-aligned content with right-side visual accents (chakra symbols, illustrations)
- No centered layouts - asymmetric design creates visual interest

**Signature Elements:**
1. **Chakra Circle Indicators** - Small, elegant circles showing current day's chakra, positioned consistently
2. **Subtle Gradient Underlines** - Each day's heading has a soft gradient underline in its chakra color
3. **Meditation Icon Badges** - Small badges indicating "Audio," "PDF," "Journaling" with soft backgrounds

**Interaction Philosophy:**
- Hover states reveal subtle color shifts (not bold changes)
- Smooth scroll transitions between days
- Clicking a day expands/reveals more details (progressive disclosure)
- Audio and PDF links have gentle hover effects

**Animation:**
- Entrance animations: Fade-in + slight upward movement as user scrolls to each day
- Hover states: Subtle color shift (chakra color becomes slightly more saturated)
- Smooth transitions between states (300-500ms)
- Avoid flashy animations - keep it meditative

**Typography System:**
- **Display Font:** "Playfair Display" or "Cormorant Garamond" (elegant serif for day titles)
- **Body Font:** "Lato" or "Open Sans" (clean sans-serif for body text)
- **Hierarchy:** Large, elegant titles for days; smaller, readable body text
- **Accent:** Small caps for chakra names (Sanskrit names)

---

## Rationale
This approach balances spiritual authenticity with modern design sensibilities. The asymmetric layout and generous spacing create a meditative experience, while the chakra colors provide visual continuity and meaning. The design feels premium without being pretentious, welcoming users into a genuine healing journey.
